# Draft: 奕策（YiCe）项目规划

## 用户需求确认

### 核心目标
创建一个基于LangGraph的开源金融分析Agent系统《奕策（YiCe）》，具备成为GitHub爆款项目的潜力。

### 关键特性
1. **多维度分析Agent团队**
   - 技术面分析师Agent
   - 基本面分析师Agent
   - 消息面分析师Agent
   - 情绪面分析师Agent

2. **数据源支持**
   - Tushare、Akshare、聚宽JQData等金融数据API
   - 用户可自定义配置数据源
   - 联网搜索MCP（Exa、Tavily等）获取最新资讯
   - 社交媒体爬取（东方财富股吧、同花顺社区、淘股吧等）- 后期付费功能

3. **技术指标计算套件**
   - MA、KDJ、RSI、BOLL等几十种常见技术指标
   - 基于历史K线、成交量数据计算

4. **AHP层次分析法决策**
   - 各分析师输出1-9标度的多空程度
   - 宏观经济/市场分析师Agent赋予权重
   - 通过AHP方法计算最终得分
   - 风险控制交易员做总结和交易建议

5. **前端界面**
   - Vue框架（用户明确偏好）
   - 仿TradingView风格K线看板
   - Agent工作流输出展示

### 开发阶段策略
- 初期专注A股市场
- 国际市场和加密货币后续支持
- Agent工作流设计可灵活调整，初期不过度投入

### 技术栈偏好（用户回答）
- **开发优先级**：核心架构 + Agent工作流基础
- **后端**：Python + FastAPI
- **开源策略**：MIT/Apache-2.0完全开源

## 研究发现

### AHP层次分析法在金融领域的应用
- AHP方法将复杂问题层次化，形成目标-准则-指标结构
- 1-9标度将专家判断数值化
- 一致性检验CR=CI/RI&lt;0.1即可接受
- 在股票投资决策中有实际应用案例

### TradingAgents项目分析
- 49.8k Stars，9k Forks，非常成功的开源项目
- 使用LangGraph构建
- 架构：分析师团队 → 多空辩论 → 风险控制 → 交易决策
- 最新版本v0.2.3（2026-03）
- 支持多语言、多LLM提供商（GPT-5.4、Gemini 3.1、Claude 4.6等）

### LangGraph最新版本和最佳实践
- **最新版本**: langgraph==1.1.6（2026年4月3日）
- **关键新特性**: 持久化执行、人在回路、全面内存管理、增强的执行信息
- **金融领域最佳实践**: 状态持久化与审计追踪、容错与降级策略、结构化输出、人在回路审批
- **与Vue集成**: @langchain/vue@0.2.0已发布，支持SSE流式输出

### 2026年前端技术栈
- **Vue**: 3.5.32（2026年4月最新）
- **状态管理**: Pinia（官方推荐，Vuex已维护模式）
- **K线图库**: Lightweight Charts（TradingView官方）
- **CSS框架**: Tailwind CSS v4（Oxide引擎）或 UnoCSS
- **构建工具**: Vite 8+（Rolldown集成）

## 开发阶段划分

基于项目规模和复杂度，建议分5个阶段开发：

**Phase 1: 项目脚手架 + 核心架构** (当前规划)
- 项目初始化、目录结构、技术栈搭建
- LangGraph基础工作流框架
- 数据源抽象层设计

**Phase 2: 数据源对接 + 技术指标计算**
- Tushare/Akshare/JQData对接
- 技术指标计算套件
- 数据缓存层

**Phase 3: Agent团队开发**
- 技术面/基本面/消息面/情绪面分析师Agent
- AHP决策模块
- 宏观经济分析师Agent

**Phase 4: 前端界面开发**
- Vue项目初始化
- K线看板（Lightweight Charts）
- Agent工作流可视化

**Phase 5: 集成测试 + 优化**
- 端到端测试
- 性能优化
- 文档完善
